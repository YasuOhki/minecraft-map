# Minecraft マップビューアー

断片的なMinecraft地図を統合表示し、座標から位置を指定できるWebアプリケーションです。

## 機能

- 複数の地図画像を1枚の大きな地図として統合表示
- X・Z座標を入力して現在位置をマーカー表示
- 地図上でクリックするとコンソールにワールド座標を表示

## 使い方

### 1. ローカルサーバーの起動

ブラウザのCORS制限のため、HTTPサーバー経由でアクセスする必要があります。

**Python 3の場合:**
```bash
python3 -m http.server 8000
```

**Node.jsの場合:**
```bash
npx serve
```

### 2. ブラウザでアクセス

```
http://localhost:8000
```

### 3. 座標を入力

- X座標とZ座標を入力
- 「位置を表示」ボタンをクリック
- 地図上に赤いマーカーが表示され、その位置まで自動スクロール

## 地図の拡張方法

### ファイル名規則（必須）

地図画像は次の形式で命名してください。**オフセット情報がないファイルは地図に含まれません**（スキップされます）。

- **形式**: `#[数値]_[右方向が+]_[下方向が+].png`
- **例**: `#17_0_0.png`（基準）、`#11_1_2.png`（#17 から右に1・下に2）

### 新しい地図タイルを追加する

1. **画像を配置**
   - `map/` に `#[番号]_[オフセットX]_[オフセットZ].png` 形式で保存
   - 基準タイル #17 を (0, 0) とした相対位置でオフセットを指定

2. **layout を自動生成**
   - プロジェクトルートで以下を実行すると、`map-config.json` の `layout` が上書きされます：
   ```bash
   node generate-map-config.js
   ```

オフセット付きでないファイル（例: `#26.png`）は「地図が存在しない」として除外され、コンソールに警告が出ます。

## 座標系の仕様

- **基準点**: #17.pngの左上 = ワールド座標 (X:0, Z:0)
- **タイルサイズ**: 180×180ピクセル
- **スケール**: 1Minecraftブロック = 1ピクセル（調整可能）

## カスタマイズ

### タイルサイズの変更

実際の画像サイズに合わせて `map-config.json` の `tileSize` を調整してください。

### スケールの調整

`script.js` の `worldToPixel` 関数内の `scale` 変数を変更：

```javascript
const scale = 2; // 1ブロック = 2ピクセル
```

### マーカーの見た目

`script.js` の `drawMarker` 関数を編集してマーカーの色やサイズを変更できます。

## トラブルシューティング

### 地図が表示されない

- HTTPサーバーが起動しているか確認
- ブラウザの開発者ツール（F12）でコンソールエラーを確認
- 画像ファイルのパスが正しいか確認

### マーカーが表示されない

- 入力した座標が地図の範囲内にあるか確認
- ブラウザのコンソールでエラーがないか確認

### 座標がずれている

- `map-config.json` の `tileSize` が実際の画像サイズと一致しているか確認
- 基準点（#17）の位置が正しいか確認
